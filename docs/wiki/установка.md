# Установка

## 1. Клонировать репозиторий

```bash
git clone git@github.com:napadayte/OpenCode.git ~/.config/opencode
```

> [!note]
> Канонический путь — `~/.config/opencode`. Скрипты сами находят корень репозитория, поэтому работают из любой папки.

## 2. Установить скрипты

```bash
~/.config/opencode/tools/bin/install-opencode-tools
```

Это сделает симлинки всех скриптов в `~/.local/bin/`.

## 3. Проверить PATH

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

Проверка:

```bash
which new-agent-workspace
which new-agent-project
```

## 4. Создать первый workspace

```bash
new-agent-workspace my-first
cd ~/code/my-first
opencode
```

Внутри OpenCode:

```
/analyze
```

Этого достаточно для первого знакомства.
