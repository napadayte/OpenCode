# Agents

------------------------------------------------------------

## Purpose

Agents describe who should handle a task.

Agents are roles.

They do not replace rules or skills.

------------------------------------------------------------

## Agent List

- brainstormer
- planner
- devil
- reviewer
- security-reviewer
- data-operator
- automation-operator
- git-operator
- docs-writer
- light-code-helper

------------------------------------------------------------

## Rule

Use agents to separate thinking roles.

Do not create many agents without a clear repeated need.
