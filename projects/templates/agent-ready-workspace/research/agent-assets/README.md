# Agent Assets Inbox

------------------------------------------------------------

## Purpose

Use this folder for captured external agent, skill, workflow, or rule ideas.

Raw captured sources are local working material.

Do not blindly promote raw external text into the system.

------------------------------------------------------------

## Intake Flow

1. Capture the source.
2. Review it.
3. Extract useful agents, skills, or rules.
4. Rewrite in our style.
5. Add only what is useful.
6. Check git diff.
7. Commit manually.

------------------------------------------------------------

## Rule

Do not commit secrets, credentials, private transcripts, or copyrighted bulk source text.

Keep final promoted content short, rewritten, and useful.
