# Skills

------------------------------------------------------------

## Purpose

Skills describe how to perform a repeatable task.

Skills are procedures.

They do not replace agents or safety rules.

------------------------------------------------------------

## Skill List

- brainstorm
- planning
- security-check
- git-safe-commit
- data-inventory
- backup-restore
- n8n-workflow-doc
- document-writing
- light-code-script
- stock-scanner-research

------------------------------------------------------------

## Rule

Add a skill only when the procedure will be reused.

Each skill should include:

- purpose
- when to use
- inputs needed
- steps
- output format
- safety rules
